# 思想星图 · Atlas of Ideas

一个以风格化3D地球为主体的世界哲学互动叙事项目。当前交付是正式制作前的8节点垂直样片，验证三件事：内容能否被空间化、关系是否可核验、电影化地球是否能在桌面和移动端保持可用。

## 当前范围

- 8位样片人物：孔子、庄子、佛陀、龙树、亚里士多德、阿维森纳、康德、法农
- 5段、约69秒的样片故事
- 故事模式与自由探索模式
- 3类示范关系：直接影响、传统延展、主题共鸣
- 人物、著作、地点、关系证据与学术来源
- 搜索、问题筛选、时间轴、两人比较、稳定公开路径
- WebGL2地球与完整文字替代入口

30人完整版在样片完成产品、内容、视觉和性能验收后再扩展，不以未审核数据占位。

## 内容治理

核心数据位于 `app/_data/atlas.ts`，构建时执行引用完整性校验。所有正式关系必须满足：

1. 至少绑定一条学术来源及定位说明；
2. 起点和终点均为有效人物；
3. 主题共鸣为非方向性关系；
4. 主题相似不得升级为直接影响；
5. 不确定地点、年代和著作归属必须标注。

详细规则见 `docs/CONTENT_POLICY.md`。

## 技术基线

- React 19、TypeScript、Vite/Vinext静态部署结构
- Three.js r184、React Three Fiber 9、WebGL2
- GSAP负责3D镜头，Motion负责DOM微交互
- Zustand共享故事与探索状态
- world-atlas与TopoJSON生成球面历史空间轮廓
- Cloudflare Sites静态托管
