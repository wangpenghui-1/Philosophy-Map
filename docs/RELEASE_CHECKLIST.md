# 8人样片公开演示版验收记录

更新时间：2026-07-15

## 自动化闸门

- ESLint、TypeScript和生产构建通过。
- 6项服务端、数据完整性、404、媒体文件和500 KB分包预算测试通过。
- 16项桌面／移动端浏览器测试通过；覆盖故事控制、搜索焦点、筛选与URL恢复、人物比较、文字探索、WebGL2降级、减少动态效果和视觉快照。
- 360、390、768、1024、1440像素宽度均无页面横向溢出，头部核心控件保持可见。
- 3D客户端代码已拆分，最大单块约364 KB；非3D应用壳未超过改造前基线。
- 本地浏览器测得桌面LCP约2.24秒、移动端LCP约0.24秒，均低于本轮2.5／4秒目标；数值仅作为当前构建基线。

## 浏览器与内容检查

- Chromium桌面与Pixel 5移动端模拟通过。
- macOS Safari已验证3D地球、8个人物节点、默认1961年时间状态、搜索和人物详情。
- 8条Stanford Encyclopedia来源返回200；同行评审论文DOI正常解析到出版方，出版方拒绝自动化访问但不影响浏览器打开。
- 8位人物的完整图、缩略图、替代文本、裁切焦点和艺术化形象说明齐全。

## 发布阻断条件

- 主题共鸣继续使用非方向性虚线与文字标签，不依赖颜色单独区分。
- 自动故事可暂停，减少动态效果模式可操作。
- WebGL2不可用或上下文丢失时自动打开完整文字探索，并提供3D重试入口。
- 搜索、人物、关系来源、比较和文字探索在移动布局中均有入口。

## 已知非阻断项

- 尚未连接可远程操控的真实手机；发布后需在一台iOS或Android真机复核触控、后台恢复、温升和持续帧率。
- 软件渲染环境不能代表真实GPU帧率；正式30人版前应在目标设备补充稳定的帧率、内存和长任务遥测。
