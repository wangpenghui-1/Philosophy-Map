# 世界哲学知识库120人物版本验收清单

更新时间：2026-07-18

## 自动化闸门

- ESLint、TypeScript和生产构建通过。
- Zod内容生成、ESLint、TypeScript和生产构建通过。
- 数据测试确认公开总数精确为120位人物、27条既有关系和44个来源，全部旧slug稳定。
- 120人发布清单包含原有30人及三批90名新增人物；非published内容不会进入公开生成物。
- 服务端与浏览器测试覆盖四类知识详情页、知识库筛选URL恢复、人物比较、文字探索、WebGL2降级、减少动态效果和视觉快照。
- 360、390、768、1024、1440像素宽度均无页面横向溢出，头部核心控件保持可见。
- 3D客户端代码已拆分，最大单块约364 KB；非3D应用壳未超过改造前基线。
- 完整审核以本轮本地桌面和移动浏览器回归报告为准；旧版本性能数值不作为120人物版本结论。

## 浏览器与内容检查

- Chromium桌面与Pixel 5移动端模拟通过。
- 人物、概念、传统和著作页面均由服务端渲染，无效slug返回404。
- 原有30位人物继续使用带授权与真实性说明的图像；新增90人使用明确标注的中性占位，不把未经核实形象当作历史肖像。
- 3D标签层在120人正式数据及240人压力输入下继续遵守桌面最多36、移动端最多16的密度预算。

## 发布阻断条件

- 主题共鸣继续使用非方向性虚线与文字标签，不依赖颜色单独区分。
- 自动故事可暂停，减少动态效果模式可操作。
- WebGL2不可用或上下文丢失时自动打开文字探索，并提供完整知识库与3D重试入口。
- 搜索、人物、关系来源、比较和文字探索在移动布局中均有入口。

## 已知非阻断项

- 尚未连接可远程操控的真实手机；发布后需在一台iOS或Android真机复核触控、后台恢复、温升和持续帧率。
- 软件渲染环境不能代表真实GPU帧率；线上部署前仍应在目标设备补充稳定的帧率、内存和长任务遥测。
- 新增90人均为索引级入口，不应被描述为标准或深入长文；外部学者复核不属于自动审核结论。
